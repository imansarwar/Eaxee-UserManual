Administration Last updated: 2022-10-27

The EA Administration module exists as the control module for the Eaxee that allows you to manage all the user, license and repository opertions. Once you have selected Eaxee Administration module you now have three options available:

User Management

The User management module involves defining and managing users within a single panel by creating a new user, removing a user, activating or deactivating a user and allowing the user to update password.

User management has two available modules user registration and password management.


Creating A User
  • Go to user management.
  • Go to user registration.
  • Enter full name.
  • Enter User ID (username; through which you can login). The entered username must be unique and must contain minimum 5 characters.
  • Enter email with correct format i.e. abc@gmail.com and click on Add button.

Success message of User Added will be displayed and user will be added in Registered Users table. The registered user can now be allocated a license and can be added to pool.


Removing A User
  • Go to user management.
  • Go to user registration.
  • Go to Registered users table.
  • Select any user by clicking on it.
  • Click on Remove button.

Removing a user will delete the user permenantly.


Activating / Deactivating A User
  • Go to user management.
  • Go to user registration.
  • Go to Registered users table.
  • Select any user by clicking on it.
  • Click on Activate / Deactivate button.

Once Deactivate button is pressed it changes to Activate button and the deactivated user will not be able to login.


Changing / Updating A Password

  • Go to user management.
  • Go to password management.
  • Go to Registered users table.
  • Select any user by clicking on it.
  • Click on change button.
  • Enter password and confirm password. Password must be of minimum 8 characters.
  • Click on update.

Success message of Password Successfully Updated is displayed.

Note

Password fields are initially disabled. Clicking on the Change button will make Password and confirm password fields active.


License Management

The license management module is an asset management that allows the named and concurrent users to manage their licenses and pool quickly and effectively.

License management has two available modules named and concurrent user


Allocating A License To Named User
  • Go to license management.
  • Go to Named user.
  • Go to available license table.
  • Select available license from Available Licenses table.
  • Select user from Activated Users table.
  • Click on Allocate.

User is successfully allocated a license for throughout the session (i.e. even if the user logouts the license remain allocated to that user). User is displayed in Allocated Licenses table.


Deallocating A License From Named User
  • Go to license management.
  • Go to Named user.
  • Select user from Allocated Licenses table.
  • Click on Deallocate.

Success message is displayed of Deallocated. License is released and is now displayed in Available Licenses table.


Note

Concurrent user licenses are those which are released only when the user session ends (i.e. once the user logouts) or any other pool module is selected to be shown.


Creating / Adding A New Pool
  • Go to license management.
  • Go to concurrent user.
  • Enter pool name (must be unique).
  • Select pool role from available roles that are of architect, repository admin and organizational portal.
  • Click on Add pool.

New user is successfully added in the table and all the licenses available of the role are displayed and activated users are displayed.


Note

Architect Role has 3 licenses. Repository Admin has 1 license. Organizational Portal has 10 licenses.


Removing A License Pool
  • Go to license management.
  • Go to concurrent user.
  • Select a pool from a table.
  • Click on Remove pool.

Adding License To A Pool
  • Go to license management.
  • Go to concurrent user.
  • Select a user from a table.
  • Click on Show pool.
  • Go to available licenses table.
  • Select license.
  • Click on Add license.

Removing License From A Pool
  • Go to license management.
  • Go to concurrent user.
  • Select a user from a table.
  • Click on Show pool.
  • Go to available licenses table.
  • Select license.
  • Click on Remove license.

Adding User To A Pool
  • Go to license management.
  • Go to concurrent user.
  • Go to activated users table.
  • Select user.
  • Click on add user.
  • Click on save button.

Removing User From A Pool
  • Go to license management.
  • Go to concurrent user.
  • Go to activated users table.
  • Select user.
  • Click on add user.
  • Click on save button.

It basically provides the accessibility of all objects, models, attachments dashboard at a single location to be cleared out at once. The steps for clearing out are as follows:


Clearing All Objects
  • Go to repository management.
  • Click on object button.

Note

The steps for clearing is same for model, attachments and dashboard.

ImpEx

Eaxee ImpEx comprises of three distinctive options: Export template option that downloads an empty excel file constituting all the objects. Export data option that downloads the bulk data of the website onto the local machine of the user in the form of excel file. Import data option lets the user add-on bulk data from their local machine to the website with fease. ImpEx easily synchronizes the data with the available meta-objects on the website.


Export Template

Selecting this option will automatically download an empty excel file of all the objects onto your local machine.

  • Click on the Export template.

Message of the empty template file being prepared is displayed on the side panel. An empty template file automatically downloads onto your local machine.


Export Data

Selecting this option will automatically download the bulk object record of the website onto your local machine.

  • Click on the Export data.

Messages of records being fetched are displayed on the side panel. Excel file automatically downloads onto your local machine.


Import Data

Selecting this option will automatically upload the bulk object record from your local machine onto the website.

  • Click on Choose file.
  • Select the file from the directory.
  • Click on import.

Success message of all data uploaded is displayed and the object records will be added in the enterprise architecture module.

Enterprise Architecture

Enterprise Architecture provides a blueprint for the users to design and manage the structure and operation of organizations.

Object Tree and Filter

Filter allow the users to refine the results of their queries and display the refined results. You can search for available object, model, atatchment, navigation, roadmap or diagram and the object tree directory will open correspondingly.Enter a search term into the filter box. Search results will appear automatically under the tree.


Filtering Data To View
  • Click on Filter field.
  • Type any named data you want to search.

Note

Enterprise tab has three available options objects, models and attachments.


Object Management

The Object Management Module exists as the control module for the object creation, editing of a object or deleting it.


Creating An Object From Tree
  • Click on Enterprise tab.
  • Click on Objects tab.
  • Click on repository folder.
  • Click on Elements folder.
  • Click on any given element layer.
  • Click on Element.
  • Right click on any element type.
  • Click on new object.

Creating An Object From Quick Access
  • Click on Quick Access tab.
  • Click on object from left panel.
  • Type in details of the object.
  • Click on save.

Creating An Object In A Diagram

Object can be easily created by diagram through drag and drop from the model's tab.

  • Click on Models Tab.
  • Right click on any folder in which you want to create Diagram.
  • Click on New Diagram.
  • Enter Diagram name.
  • Click on Create.
  • Drag and drop the object selected from the panel.
  • Click on save.

Editing an object From Tree
  • Click on Objects tab.
  • Click on repository folder.
  • Click on Elements folder.
  • Click on any given element layer.
  • Click on Element.
  • Click on any element type.
  • Select an object an edit as per your preference.
  • Click on save button.

Editing an object From Quick Access
  • Click on Quick Access tab.
  • Select an object from the table an edit as per your preference.
  • Click on save button.

Editing An Object In A Diagram
  • Click on Models Tab.
  • Right click on any folder in which you want to edit Diagram.
  • Click on Diagram and edit the details as per your preference.
  • Click on save button.

Removing / Deleting An Object From Tree
  • Click on Objects tab.
  • Click on repository folder.
  • Click on Elements folder.
  • Click on any given element layer.
  • Click on Element.
  • Right click on any element type.
  • Click on remove.

Note

The steps for removing / deleting an object is same from tree, quick access and from the diagram.


Restoring An Object From Recycle bin
  • Click on Objects tab.
  • Click on recycle bin.
  • Click on Elements folder.
  • Right click on object you want to restore.
  • Click on restore.

Note

The steps for restoring an object is same from tree, quick access and from the diagram.


Permenantly Deleting An Object From Recycle bin
  • Click on Objects tab.
  • Click on recycle bin.
  • Click on Elements folder.
  • Right click on object you want to restore.
  • Click on delete.
  • Click on confirm to permenantly delete the object.

Note

The steps for deleting an object is same from tree, quick access and from the diagram.

File Management

File management module works by moving all your documents into the digital space. Existing computer files are easily integrated into the website and objects can be easily change as per perference.


Uploading A File
  • Click on Attachments tab.
  • Right click to create a new folder.
  • Right click on the folder that is created and click on new attachment.
  • Click on choose file and select the file from the directory.
  • Click on upload

Attaching A File To An Object
  • Click on Objects tab.
  • Click on repository folder.
  • Click on Elements folder.
  • Click on any given element layer.
  • Click on Element.
  • Click on any element type.
  • Select an object and select attachmnets from the left panel.
  • Select the attachment and click on save changes.

Note

The attachments can also be linked to the object from the Quick Access by selecting the object and selecting the file from the attachments tab.

Switching Among The Objects / Models

The browsing history provides the list of all visited objects / models. It lets you switch among the object and models and provides you with the brief of all visited objects / modules.


Closing A Window From History Tab
  • Click on History tab.
  • Click on cross sign to close selected window.

Note

The history tab also lets you close all the objects / models windows at once by clicking on Close All .


Switching Windows From History Tab
  • Click on History tab.
  • Click on window where you want to search.

Organization Portal

An organizational portal serves as the entrance gate for organizational users and one of its main objectives is to manage the corporate knowledge; it enables us to link information, knowledge and data via incorporated charts.

Chart Management

Organization chart management provide gateways to organization data, information, and knowledge in the form of charts.

Note

Organization portal provides three chart types i.e., column chart, clustered column chart and piechart. The tab and chart names are editable.


Creating A Tab
  • Go to organization portal.
  • Click on Add button.
  • Enter the tab title and click on add button.
Tab with 6 empty charts will be created.

Note

The current version limits the creation of 9 tabs.


Text View
Text view provides the information of all the application component and their corresponding attribute / criticality in tabular form.
  • Click on text view button.
  • Select an element and its attribute.
  • Click on generate button.

Chart View
Chart view provides the information of all the application component and their corresponding attribute / criticality in chart form.
  • Click on chart view button.
  • Select an element.
  • Select attribute of the corresponding element and click on add.
  • Select an operation and where the operation is to be applied either on attribute or on element.
  • Select chart type.
  • Click on generate chart.

Creating A Column Chart
  • Click on chart view button.
  • Select Application Interface from element.
  • Select Short Name from attribute and click on add.
  • Select Count from operation.
  • Select Element from the operation applied on.
  • Select Column from chart type.
  • Click on generate chart.

Note

The steps shown above are for Application Interface element. For all other elements the steps are same for column chart creation. Two or more elements can aso be plotted on a single chart.


Creating A Clustered Column Chart
  • Click on chart view button.
  • Select Business Application from element.
  • Select Functional Adequacy from attribute and click on add.
  • Select Average from operation.
  • Select Element from the operation applied on.
  • Select Clustered Column from chart type.
  • Click on generate chart.

Note

The steps shown above are for Business Application element. For all other elements the steps are same for clustered column chart creation. Two or more elements can aso be plotted on a single chart.


Creating A Pie Chart
  • Click on chart view button.
  • Select Business Application from element.
  • Select Functional Adequacy from attribute and click on add.
  • Select Distribution from operation.
  • Select Element from the operation applied on.
  • Select Pie from chart type.
  • Click on generate chart.

Note

The steps shown above are for Business Application element. For all other elements the steps are same for pie chart creation.
